# 白音七 イラストアイコン素材 v02 generic

白音七の鑑定画面向けイラストアイコン素材です。

## v02の方針

ライフパス、パーソナルデイ、名前ナンバー、キーワード、星座など、ユーザーごとに変わる値は画像に焼き込まない構成に変更しました。

- 数値はUIテキスト側で表示
- キーワードはUIテキスト側で表示
- 星座名はUIテキスト側で表示
- 画像は「カテゴリの雰囲気」だけを担当
