# プレミアム共通権限基盤の監査記録

## 採用した判定

- 深掘り鑑定: `premium` かつ `active` かつ `deep_enabled`
- 月間音声枠: `premium` かつ `active` かつ月間残数あり
- 単発音声枠: 会員プランにかかわらず単発残数あり
- 音声利用権全体: 月間音声枠または単発音声枠のどちらかが利用可能

旧 `stripe_*` 属性は権限判定に使用しない。

## 今回開放しない機能

- 深掘り鑑定の開始
- 鑑定本文のTTS生成
- 閉鎖中の音声アップロード画面

共通判定は表示整理にのみ使用し、各機能のサーバー側権限制御が完成するまで開始CTAを表示しない。

## 旧プラン変更API

`src/pages/members.astro` には旧 `subscription/change-plan` APIへの処理が残っている。
現在の `public/js/billing.js` は常に課金操作を無効化する互換シムであり、この経路から旧APIは呼び出されない。
MOSH移行後の変更・解約運用が確定するまでは復活させず、後続作業で削除可否を判断する。

## Lambdaの責務

`lambda/voice_upload.py` と `lambda/lambda_function.py` はユーザー音声アップロード鑑定の処理である。
鑑定本文のTTS生成とは別責務のため、今回変更していない。
