# Free Personal AI Context Pack Template v0.1

**Stop starting every AI chat from zero.**

**A reusable context template for people who use AI for real work.**

Use this fill-in template to organize the background you regularly give to ChatGPT, Claude, Gemini, or another AI tool.

You control the document. You update it when your work changes. You paste the relevant version when you need it.

**Not a memory app. Not another prompt dump. A practical starter kit for giving AI the context it actually needs.**

> This is a reusable text template. It does not create permanent AI memory, install software, or sync information between tools.

### How to Use This Template

1. Replace the bracketed placeholders with short, useful answers.
2. Keep only the information that changes how an AI should help you.
3. Remove sections that do not apply.
4. Review the Sensitive Information Reminder before pasting the pack into any AI tool.
5. Paste the completed pack at the start of a new chat, then add your actual request.

## 6. AI Rules

*Set practical working preferences. These rules do not override platform policies or remove the need for human review.*

### Response Rules

- Lead with the answer, recommendation, or useful result.
- Use this Context Pack only when it is relevant to the current request.
- Separate facts, assumptions, and recommendations when the distinction matters.
- Do not repeat the Context Pack back to me unless I ask.
- [Add your own response rule]

### Uncertainty Rules

- Say what is unknown instead of inventing missing details.
- Flag assumptions that could materially change the answer.
- Suggest a way to verify important claims.
- [Add your own verification rule]

### Question Rules

- Ask a focused question when missing information would materially change the result.
- Do not ask broad discovery questions when a reasonable, reversible assumption will keep the work moving.

### Challenge Rules

- Explain the concern and offer a practical alternative.

## 10. Sensitive Information Reminder

Before using this template, remove anything that should not be shared with the selected AI tool.

### Do Not Include

- Passwords, API keys, authentication tokens, or private access links
- Private customer records or identifying customer data
- Confidential contracts, internal documents, or protected business information
- Legally protected or regulated information
- Personal identifiers that are not required for the task
- Any workplace information that is not approved for use with the selected tool

### Before You Paste

- Confirm that you are authorized to use the information.
- Follow your organization’s approved AI and data-handling rules.
- Replace real identifiers with neutral labels when possible.
- Share the minimum context needed for the task.
- Review important output before using, publishing, or acting on it.

> This template does not make an AI retain information permanently. It does not create perfect answers or replace your judgment.
